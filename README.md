# travis-cli

## Install

```
$ [sudo]npm i -g 
```

## Usages

```
$ travis -h

  Usage: travis [options]

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
    -i, --init     init default .travis.yml
    -b, --badge    get badge for README.md
```

create .travis.yml

```
travis -i
```

get badge for README.md


```
travis -b
```
